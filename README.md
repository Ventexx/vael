# Vanta

A beautifully minimal drag-and-drop task app with a refined dark aqua aesthetic.

## Features
Add tasks instantly
Drag and reorder tasks
Spacer separators for categories
Inline task editing
Undo / redo support
Smooth minimal UI
No setup required
Single HTML file
No persistent storage

## Running

Simply open ```index.html``` in your browser.

## Notes
- Tasks reset on refresh
- Designed for fast, distraction-free task management
- Runs completely locally
